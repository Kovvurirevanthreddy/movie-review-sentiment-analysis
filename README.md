🎬 Movie Review Sentiment Analysis

📌 Description
This project performs sentiment analysis on movie reviews using Natural Language Processing (NLP) and Machine Learning techniques. It classifies reviews as **positive** or **negative** using the IMDB 50K dataset.

🚀 Features
- Text preprocessing (lowercasing, HTML removal, punctuation cleaning)
- Stopword removal using NLTK
- Feature extraction using:
  - CountVectorizer
  - TF-IDF Vectorizer
- Model training using Naive Bayes
- Performance evaluation (Precision, Recall, F1-score)
- WordCloud visualization for positive and negative reviews
- Probability-based analysis (Naive Bayes theory)

🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK
- Matplotlib

📂 How to Run
Open the notebook.ipynb file in Google Colab or Jupyter Notebook and run all cells.

📂 Dataset
IMDB 50K Movie Reviews Dataset (from Kaggle)
